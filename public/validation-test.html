About This Tool

This is a test utility for the API validation framework. It allows you to test:

  • Basic Validation - Schema-based data validation using Zod
  • Security Validation - Detection of malicious inputs like SQL injection
  • Health Check - Verify the validation server is running

How It Works

The validation server runs on port 4000 and provides endpoints that can be used to test data validation:

  • GET /api/health - Server health check
  • POST /api/validate/basic - Schema validation for user data
  • POST /api/validate/security - Security validation for user inputs

Testing

You can also run server-side validation tests using the following scripts:

  • ./start-api.sh - Start the validation server
  • ./run-api-validation-test.sh - Run automated tests
